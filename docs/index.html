Chat 모드

AI와 대화하고 질문하세요

  • • LangGraph 워크플로우 (6가지: Instant, Sequential, Deep, Coding, RAG, Browser)
  • • RAG 문서 검색 (편집, 폴더 관리, Export/Import)
  • • MCP 도구 통합 (GitHub, Brave Search, Filesystem 등)
  • • 이미지 생성 및 해석 (ComfyUI, Vision API)
  • • Persona 시스템 (AI 역할 정의, SQLite 영구 저장)
  • • Quick Question (빠른 질문, 최대 5개 단축키)
  • • GitHub Sync (Token 기반, AES-256-GCM 암호화)

🤖 핵심 기능

LangGraph 워크플로우

다양한 사고(Thinking) 모드를 지원하는 LangGraph 기반 워크플로우.
Instant, Sequential, Tree-of-Thought, Deep Thinking 등 상황에 맞는 최적의 응답 생성.

  • Chat, RAG, Agent, Coding Agent 그래프 지원
  • 실시간 스트리밍으로 사고 과정 시각화
  • conversationId 기반 격리로 안정적인 동시 대화

🎭 NEW in v0.6.5

AI Agents & Persona

목적에 특화된 도구형 에이전트와 유연한 페르소나 시스템.
코딩부터 프레젠테이션 제작까지 전문적인 작업을 수행합니다.

  • PPT Agent: 프레젠테이션 생성, 슬라이드 수정, 디자인 변경, 다국어 번역
  • Editor Agent: 파일 시스템 제어, 터미널 실행, 복잡한 리팩토링 수행
  • Persona: 역할 정의 (개발자, 번역가 등) 및 커스텀 페르소나 영구 저장
📚 문서 검색

RAG (검색 증강 생성)

강력한 문서 검색과 임베딩 기반 RAG 시스템.
텍스트, URL, 파일(PDF, DOCX, TXT, MD) 업로드 지원 및 LLM 기반 문서 편집 기능.

  • SQLite-vec, OpenSearch, Elasticsearch, pgvector 지원
  • 문서 편집 AI 기능 (정제, 확장, 축약, 검증, 커스텀 프롬프트)
  • 폴더 구조 관리 (드래그 앤 드롭, Tree/List/Grid 뷰)
  • Team Docs: 여러 GitHub 레포지토리 연동 및 팀 문서 지식 공유
  • Export/Import (JSON 형식, 중복 처리, 백업/복원)

🌐 NEW in v0.6.0

브라우저 자동화 (27개 도구)

Electron BrowserView 기반 Chromium 브라우저 통합.
Vision 기반 UI 제어 및 Google Search Tools로 Perplexity 수준의 검색 지원.

  • DOM Inspection: 의미론적 구조 분석, 자연어 요소 검색
  • Vision Tools: Set-of-Mark 주석, 좌표 클릭, 마커 클릭
  • Google Search: 웹/뉴스/Scholar/이미지, 고급 필터 (날짜, 언어, 사이트)
  • Bot 감지 우회: Stealth 핑거프린트, 자연스러운 타이밍, 헤더 강화
🛠️ 도구 통합

MCP 프로토콜

Model Context Protocol을 통한 표준화된 도구 및 컨텍스트 통합.
Human-in-the-loop 도구 승인으로 안전한 실행.

  • GitHub, Brave Search, Git, Filesystem 등 템플릿 제공
  • 환경 변수 설정 UI로 간편한 구성
  • 도구 실행 전 사용자 승인 (5분 타임아웃)

📁 assets/videos/mcp-tools.gif

🔄 NEW in v0.6.5

GitHub Sync (클라우드 동기화)

GitHub Personal Access Token을 사용한 안전한 데이터 동기화.
설정, 문서, 페르소나, 이미지, 대화 내역을 여러 기기에서 공유.

  • AES-256-GCM 암호화로 API 키, 토큰 등 민감 정보 보호
  • 데이터 타입별 선택적 동기화 (설정/문서/페르소나/이미지/대화)
  • 개별/전체 동기화, 상태 추적, 연결 테스트

빠른 시작

5분 안에 시작하세요

1

다운로드 및 설치

플랫폼에 맞는 설치 파일을 다운로드하고 설치하세요.

# Windows: SEPilot-Setup-0.6.5.exe
# macOS: SEPilot-0.6.5.dmg
# Linux: SEPilot-0.6.5.AppImage
2

LLM 설정

왼쪽 하단 설정 아이콘을 클릭하고 LLM 제공자와 API 키를 설정하세요.

  • OpenAI (gpt-4o, gpt-4-turbo, gpt-3.5-turbo)
  • Anthropic (claude-3-5-sonnet, claude-3-opus)
  • Google (gemini-pro, gemini-1.5-pro)
  • Custom (OpenAI-compatible API)
3
