🤖 핵심 기능

LangGraph 워크플로우

다양한 사고(Thinking) 모드를 지원하는 LangGraph 기반 워크플로우.
Instant, Sequential, Tree-of-Thought, Deep Thinking 등 상황에 맞는 최적의 응답 생성.

  • Chat, RAG, Agent, Coding Agent 그래프 지원
  • 실시간 스트리밍으로 사고 과정 시각화
  • conversationId 기반 격리로 안정적인 동시 대화

📚 문서 검색

RAG (검색 증강 생성)

강력한 문서 검색과 임베딩 기반 RAG 시스템.
텍스트, URL, 파일(PDF, DOCX) 업로드 지원 및 LLM 기반 문서 정제 기능.

  • SQLite-vec, OpenSearch, Elasticsearch, pgvector 지원
  • 다양한 문서 형식 지원 (TXT, MD, PDF, DOCX)
  • URL에서 자동 마크다운 변환 및 LLM 정제
🛠️ 도구 통합

MCP 프로토콜

Model Context Protocol을 통한 표준화된 도구 및 컨텍스트 통합.
Human-in-the-loop 도구 승인으로 안전한 실행.

  • GitHub, Brave Search, Git, Filesystem 등 템플릿 제공
  • 환경 변수 설정 UI로 간편한 구성
  • 도구 실행 전 사용자 승인 (5분 타임아웃)

빠른 시작

5분 안에 시작하세요

1

다운로드 및 설치

플랫폼에 맞는 설치 파일을 다운로드하고 설치하세요.

# Windows: SEPilot-Setup-0.5.0.exe
# macOS: SEPilot-0.5.0.dmg
# Linux: SEPilot-0.5.0.AppImage
2

LLM 설정

왼쪽 하단 설정 아이콘을 클릭하고 LLM 제공자와 API 키를 설정하세요.

  • OpenAI (gpt-4, gpt-4-turbo, gpt-3.5-turbo)
  • Anthropic (claude-3-5-sonnet, claude-3-opus)
  • Custom (OpenAI-compatible API)
3
