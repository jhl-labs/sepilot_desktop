# Release Notes - v0.6.12

**출시일**: 2024-12-19

## 🌍 다국어(i18n) 지원 확대

### 새로 추가된 다국어 지원 컴포넌트

#### Chat 모드

- **SidebarChat**: 사이드바 채팅 버튼 툴팁
  - AI 페르소나 관리
  - 문서 관리
  - 이미지 갤러리
  - Presentation 모드
  - 설정

#### Editor 모드

- **FileExplorer**: 파일 탐색기
  - 디렉토리 선택
  - 새 파일/폴더 생성
  - 툴팁 및 메시지
  - 빈 상태 메시지
  - 다이얼로그 (파일/폴더 생성)

- **FileTreeContextMenu**: 파일 트리 컨텍스트 메뉴
  - 파일/폴더 작업 (복사, 잘라내기, 붙여넣기)
  - 경로 복사 (절대 경로, 상대 경로, 파일 이름)
  - 시스템 통합 (탐색기에서 열기, 터미널에서 열기, 기본 앱으로 열기)
  - 편집 작업 (이름 변경, 삭제)
  - 폴더 작업 (검색, 하위 폴더 접기)

#### Browser 모드

- **SnapshotsList**: 스냅샷 관리
  - 제목, 로딩 상태
  - 빈 상태 메시지
  - 확인 대화상자
  - 에러 메시지

- **BookmarksList**: 북마크 관리
  - 폴더 관리 (추가, 삭제)
  - 북마크 관리 (추가, 열기, 삭제)
  - 확인 대화상자
  - 성공/에러 메시지

- **BrowserAgentLogsView**: Agent 실행 로그
  - 페이즈 라벨 (사고 중, 도구 호출, 도구 결과, 결정, 완료, 오류)
  - 상태 메시지
  - 버튼 텍스트 (뒤로 가기, 로그 삭제)
  - 빈 상태 메시지

- **BrowserToolsList**: Browser Agent 도구 목록
  - 27개 도구 설명
  - 카테고리 설명 (Navigation, Page Inspection, Page Interaction, Tab Management, Vision-based Tools, Google Search Tools)
  - 안내 메시지

### 지원 언어

- **한국어** (ko): 전체 지원
- **영어** (en): 신규 번역 추가
- **중국어** (zh): 신규 번역 추가

## 📝 번역 키 구조

```
app.sidebarChat.*          - 사이드바 채팅 툴팁
fileExplorer.*             - 파일 탐색기 및 컨텍스트 메뉴
browser.snapshots.*        - 스냅샷 관리
browser.bookmarks.*        - 북마크 관리
browser.agentLogs.*        - Agent 실행 로그
browser.toolsList.*        - Browser Agent 도구 목록
```

## 🔧 기술적 개선

- 모든 하드코딩된 한국어 문자열을 i18n 번역 키로 변경
- `react-i18next`의 `useTranslation()` 훅 사용
- 동적 번역 키 지원 (`t('key', { variable })`)
- 컴포넌트별 번역 네임스페이스 분리

## 🎯 사용자 경험 개선

- 사용자 언어 설정에 따라 모든 UI 텍스트가 자동으로 변경
- 일관된 번역 품질
- 더 나은 접근성
