# SEPilot Desktop v0.7.2 릴리스 노트

**릴리즈 날짜**: 2026-01-05

---

## ✨ 새로운 기능

### 🌍 다국어 지원 확장 (i18n)

UI 전반에 걸쳐 다국어 지원이 대폭 확장되었습니다.

- **WorkingDirectoryIndicator**: 작업 디렉토리 관련 텍스트 국제화
- **Sidebar**: 새 대화 버튼 툴팁 국제화
- **DocsSyncDialog**: GitHub 동기화 설정 전체 UI 국제화
  - Personal Docs / Team Docs 탭 라벨
  - Server Type, GHES URL, Token, Owner, Repository 등 모든 폼 필드
  - 저장/취소/삭제/편집 버튼 및 상태 메시지

### 🛠 Coding Agent 도구 필터링 개선

Thinking 모드에 따른 도구 표시가 개선되었습니다.

- **thinkingMode === 'coding'** 일 때만 builtin 도구(file_read, file_write, file_edit, file_list, command_execute, grep_search) 표시
- 일반 Chat 모드에서는 MCP 서버 도구만 표시하여 UI 간소화

## 🎨 UI/UX 개선

### 프레젠테이션 컴포넌트 리팩토링

- **PresentationChat**: 코드 구조 개선 및 가독성 향상
- **SlidePreview**: 슬라이드 미리보기 컴포넌트 최적화

## 📚 문서 업데이트

### docs/index.html 현행화

- 최신 모델명 반영: GPT-4.1, Claude 4, Gemini 2.5

- RAG 관련 설명 간소화 및 Team Docs GitHub 동기화 섹션 추가
- Vision Analysis 설명 현행화
