# SEPilot Desktop v0.7.0 릴리스 노트

**릴리즈 날짜**: 2026-01-04

**SEPilot Desktop v0.7.0**은 v0.5.0 이후 축적된 대규모 업데이트와 안정성 개선을 포함하는 메이저 마일스톤입니다. 단순한 Chat UI를 넘어, **완전한 AI IDE(Integrated Development Environment)**로 진화했습니다.

---

## 🌟 하이라이트 (v0.5.0 이후 주요 변화)

v0.5.0 이후, SEPilot은 다음과 같은 혁신적인 기능들을 도입했습니다:

### 1. 💻 Editor 모드 & IDE 기능

Chat 기반 인터페이스에서 벗어나, 실제 코드를 작성하고 수정할 수 있는 **Editor 모드**가 추가되었습니다.

- **Monaco Editor 통합**: VS Code와 동일한 강력한 코드 편집 경험 제공 (Syntax Highlighting, 미니맵, 다크모드)
- **통합 파일 탐색기**: 프로젝트 파일 트리 뷰, 파일 생성/삭제/이름변경 지원
- **통합 터미널**: xterm.js 기반의 크로스 플랫폼(PowerShell, Bash) 터미널 내장
- **Browser 모드**: 앱 내에서 웹 브라우징, 문서 참조가 가능한 내장 브라우저 (Browser Agent 연동)

### 2. 🧠 RAG 2.0 (검색 증강 생성 대폭 개선)

단순한 벡터 검색을 넘어선 차세대 RAG 엔진을 탑재했습니다.

- **Parent Document Retrieval**: 문서의 맥락을 유지하기 위해 작은 청크로 검색하고 원본 문서를 참조
- **Hybrid Search**: 벡터 유사도(Semantic)와 키워드 매칭(BM25)을 결합하여 검색 정확도 향상
- **Metadata Filtering**: 폴더, 태그, 카테고리 기반의 정교한 필터링 및 부스팅

### 3. 🎨 Creative Tools

- **AI Presentation Lab**: 대화만으로 전문적인 PPT 프레젠테이션 슬라이드 자동 생성 및 내보내기
- **ImageGen 통합**: NanoBanana API 및 ComfyUI를 통한 멀티 프로바이더 이미지 생성 지원
- **Writing Tools**: Notion 스타일의 AI 글쓰기 도구 (요약, 톤 변경, 문법 수정 등)

### 4. 🎭 AI Personas

- **페르소나 시스템**: 번역가, 시니어 개발자, 영어 선생님 등 다양한 전문가 인격 부여
- **Custom Persona**: 사용자 정의 시스템 프롬프트로 나만의 AI 비서 생성

## ✨ v0.7.0 새로운 기능

### 🧜‍♂️ Mermaid 다이어그램 Fallback 시스템

복잡한 다이어그램 생성 시 발생할 수 있는 오류를 자동으로 해결하는 지능형 시스템이 도입되었습니다.

- **자동 문법 수정 (Self-Correction)**: Mermaid 렌더링 실패 시, AI가 자동으로 문법 오류를 분석하고 수정한 뒤 재시도합니다.
- **Text Fallback 모드**: 반복적인 수정 실패 시, 다이어그램을 ASCII Art나 구조화된 텍스트로 자동 변환하여 정보를 손실 없이 전달합니다.
- **향상된 안정성**: 사용자는 더 이상 "Render Error" 메시지를 보지 않고, 항상 시각화된 결과물을 얻을 수 있습니다.

### ⌨️ 인라인 자동완성 토글

사용자의 타이핑 흐름을 방해하지 않는 스마트한 제어 기능이 추가되었습니다.

- **Autocomplete Toggle**: 설정에서 인라인 자동완성(Ghost Text) 기능의 켜고 끄기를 쉽게 제어할 수 있습니다.
- **자연어 처리 강화**: 단순 코드 완성을 넘어, 문맥에 맞는 자연어 문장 자동 완성을 지원합니다.

## 🛠 버그 수정 및 개선 사항

- **코드 품질 개선**: ESLint 규칙 준수를 위한 전면적인 코드 리팩토링
  - 중복 타입 정의 제거 (EditorToolsList.tsx)
  - 미사용 import 제거 및 코드 정리
  - 조건문에 curly braces 추가로 가독성 향상
  - Template literal 사용으로 문자열 처리 개선
  - TypeScript const 사용 최적화
- **Ruler Drag Fix**: 에디터의 Word Wrap Ruler가 드래그되지 않던 UI 문제를 해결하여 자유롭게 가이드 선을 조절할 수 있습니다.
- **Agent 안정성 개선**: 긴 작업 수행 시 에이전트가 멈추던 현상(Agent Halt)을 수정하여, 복잡한 태스크도 끝까지 완수하도록 개선했습니다.
- **Electron Stability**: `EPIPE` 에러 등 메인 프로세스와의 통신 불안정 문제를 해결하여 앱의 전반적인 안정성을 높였습니다.
- **성능 최적화**: 대용량 파일 편집 및 렌더링 속도가 최적화되었습니다.
