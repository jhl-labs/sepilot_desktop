# SEPilot Desktop v0.6.6 릴리스 노트

이번 릴리스는 **다국어(i18n) 지원**과 **커스텀 헤더 설정** 기능을 추가한 버전입니다.

## ✨ 주요 기능

### 1. 다국어(i18n) 지원 (Phase 1)

다국어 지원 인프라를 구축하고, 주요 설정 화면에 다국어 지원을 적용했습니다.

- **Settings 화면 다국어화**:
  - SettingsDialog 전체 다국어 지원
  - MCPSettingsTab 다국어 지원
  - ImageGenSettingsTab 다국어 지원
- **Quick Input 다국어 지원**: 전역 단축키 입력창 다국어화
- **i18next 라이브러리 통합**: React-i18next 기반의 확장 가능한 다국어 시스템 도입

### 2. 연결 수준 커스텀 헤더 설정

LLM Provider 연결 시 커스텀 HTTP 헤더를 설정할 수 있는 기능이 추가되었습니다.

- Provider별 독립적인 헤더 설정 가능
- 프록시 환경 및 기업 네트워크에서의 유연한 연동 지원

- `refactor`: HTTP 통신 중앙 집중화 모듈 (`lib/http/`) 생성

  - SSL 검증 설정 지원
  - WebSocket 프록시 지원 (Node.js 환경)
  - SSE(Server-Sent Events) 프록시 지원
  - 마이그레이션된 파일:
    - `lib/llm/http-utils.ts`
    - `lib/imagegen/nanobanana-client.ts`

    - `electron/ipc/handlers/auth.ts`
