# SEPilot Desktop v0.6.3 릴리스 노트

## 주요 변경 사항

### 버그 수정

- **Team Docs Push 에러 방지 검증 로직 추가**
  - Team Docs 문서 편집 시 `teamDocsId` 누락으로 인한 에러 방지
  - Push 전 `teamDocsId` 검증 로직 추가
  - 검증 실패 시 사용자에게 명확한 에러 메시지 표시
  - 에러 메시지: "Team Docs ID가 누락되었습니다. 이 문서는 Team Docs 동기화 대상이 아닙니다."
  - 코드 위치: `components/rag/DocumentEditDialog.tsx:219-226`

## 기술적 세부 사항

### 에러 발생 원인

- Team Docs 문서에 `teamDocsId` 메타데이터가 누락된 상태에서 GitHub Push 시도
- `pushDocument` IPC 호출 시 `teamDocsId: undefined` 전달
- `team-docs.ts:300`에서 설정을 찾지 못해 에러 발생

### 해결 방법

1. Push 요청 전 `teamDocsId` 존재 여부 검증
2. 검증 실패 시 Push 중단 및 사용자에게 알림
3. 불필요한 IPC 호출 방지 및 명확한 에러 핸들링

## 영향 범위

- Team Docs 문서 편집 기능
- GitHub 동기화 안정성 향상
