## ✨ 새로운 기능

### 🌍 LangGraph Agent 다국어 지원 강화

모든 LangGraph Agent와 Editor Agent가 사용자의 언어 설정을 자동으로 인식하고 해당 언어로 응답합니다.

**적용된 Agent:**

- **Deep Thinking Agent**: 심층 분석 시 사용자 언어로 답변
- **Deep Web Research Agent**: 웹 연구 결과를 사용자 언어로 제공
- **Sequential Thinking Agent**: 순차적 사고 과정을 사용자 언어로 설명
- **Tree of Thought Agent**: 사고 트리 분석을 사용자 언어로 표현
- **Editor Agent**: 코드 에디팅 제안 및 설명을 사용자 언어로 제공
- **Editor Agent Advanced**: 고급 코드 작업 시 사용자 언어로 안내

**지원 언어:**

- 한국어 (ko)
- English (en)
- 中文 (zh)

**구현 방식:**

- Main Process: 데이터베이스 설정(`app_config`)에서 언어 설정 자동 읽기
- Renderer Process: localStorage에서 언어 설정 자동 읽기
- 언어 설정에 따라 Agent의 시스템 프롬프트에 적절한 언어 지시 자동 추가

## 🛠 기술적 개선

### 언어 감지 로직

각 Agent에 다음 함수들이 추가되었습니다:

```typescript
async function getUserLanguage(): Promise<SupportedLanguage>;
function getLanguageInstruction(language: SupportedLanguage): string;
```

**작동 방식:**

1. Main Process(Electron): `databaseService`에서 `app_config` 설정 읽기
2. Renderer Process(Browser): localStorage에서 `sepilot_language` 읽기
3. 설정 없을 시 기본값 'ko'(한국어) 사용
4. 감지된 언어에 따라 적절한 언어 지시 문구를 시스템 프롬프트에 추가

**에러 처리:**

- 언어 설정 읽기 실패 시 자동으로 한국어로 폴백
- 안정적인 다국어 지원 보장
